\section{Peripherals}
\label{sec:periphs}

The Basys2 has manny peripherals options such as:
-7-segments display
-on/off switch
-push buttons
-VGA port
-PS/2 port
-6-pin headers

\subsection{Peripherals Used}

The peripherals that we will de using are the 7-segments display and the push buttons.
